# بوت القوانين التفاعلي 📜

بوت ديسكورد متقدم لإدارة القوانين بشكل تفاعلي مع لوحة تحكم كاملة.

## ✨ المميزات

### 🎯 نظام القوانين التفاعلي
- ✅ **أزرار تفاعلية** - يظهر للمستخدمين أزرار لاختيار فئة القوانين
- ✅ **استجابة فورية** - عند الضغط على زر تظهر القوانين فوراً
- ✅ **رسالة خاصة** - تظهر القوانين فقط للمستخدم الذي ضغط

### 🎨 لوحة التحكم
- ✅ **إرسال القوانين** - إرسال رسالة القوانين التفاعلية
- ✅ **إدارة القوانين** - إضافة، تعديل، وحذف الفئات
- ✅ **رفع الصور** - إضافة صور للقوانين
- ✅ **حالة مباشرة** - مؤشرات اتصال البوت و WebSocket

### 💾 نظام الحفظ
- ✅ **حفظ تلقائي** - القوانين تُحفظ في ملف JSON
- ✅ **تحميل تلقائي** - القوانين تُحمل عند بدء التشغيل
- ✅ **تعديل سهل** - يمكن تعديل القوانين من لوحة التحكم

## 🚀 التثبيت

### المتطلبات
- Python 3.7+
- Discord Bot Token

### الخطوات

1. **تثبيت المكتبات:**
```bash
pip install -r interactive_requirements.txt
```

2. **إعداد متغيرات البيئة:**
```
DISCORD_TOKEN = توكن البوت
APPLICATION_ID = معرف التطبيق
GUILD_ID = معرف السيرفر
ADMIN_PASSWORD = كلمة المرور
FLASK_SECRET = أي قيمة عشوائية
```

3. **تشغيل البوت:**
```bash
python interactive_rules_bot.py
```

## 📋 الاستخدام

### 1. إرسال القوانين التفاعلية
1. سجل الدخول إلى لوحة التحكم
2. اختر السيرفر والقناة
3. أضف صورة (اختياري)
4. اضغط "إرسال القوانين التفاعلية"

### 2. إدارة القوانين
1. اذهب إلى تبويب "إدارة القوانين"
2. **إضافة فئة جديدة:**
   - اضغط "إضافة فئة جديدة"
   - أدخل اسم الفئة
   - أدخل القوانين (كل قانون في سطر جديد)
   - اضغط "إضافة"

3. **تعديل فئة:**
   - اضغط "تعديل" بجانب الفئة
   - عدل الاسم والقوانين
   - اضغط "حفظ التعديلات"

4. **حذف فئة:**
   - اضغط "حذف" بجانب الفئة
   - أكد الحذف

## 🐛 استكشاف الأخطاء

### البوت لا يظهر متصل:
1. تأكد من صحة DISCORD_TOKEN
2. تأكد من أن البوت لديه الصلاحيات المطلوبة
3. تحقق من السجلات (logs)

### القوانين لا تُحفظ:
1. تأكد من وجود ملف rules_data.json
2. تأكد من صلاحيات الكتابة على الملف
3. تحقق من السجلات (logs)

### الأزرار التفاعلية لا تعمل:
1. تأكد من أن WebSocket متصل
2. تأكد من أن البوت لديه صلاحيات Intents
3. تحقق من السجلات (logs)
