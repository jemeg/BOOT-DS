# 🤖 دليل بوت ديسكورد المتقدم

## 🎯 **المميزات الرئيسية**

### ✅ **ما يمكن للبوت فعله:**
- 🎮 **عرض قوانين مخصصة** عند دخول الروم الصوتي
- 🎉 **رسائل ترحيب** مختلفة لكل روم
- 📢 **إعلانات مخصصة** لكل قناة
- 📝 **محتوى نصي** مخصص
- 🖼️ **صور مختلفة** لكل روم
- ⚙️ **تحكم كامل** في المحتوى لكل قناة

---

## 🚀 **طريقة التشغيل**

### 1. **الرفع على Railway**
1. ارفع كل الملفات لمشروع Railway جديد
2. أضف المتغيرات في Variables:
   - `DISCORD_TOKEN` - توكن البوت
   - `APPLICATION_ID` - معرف التطبيق
   - `GUILD_ID` - معرف السيرفر
3. اضغط Deploy

### 2. **إضافة البوت للسيرفر**
1. من Discord Developer Portal > OAuth2
2. اختر الصلاحيات:
   - `bot`
   - `applications.commands`
3. في Bot Permissions اختر:
   - `Send Messages`
   - `Embed Links`
   - `Read Message History`
   - `Connect`
   - `Speak`

## 📋 **الأوامر المتاحة**

### 🎮 **الأوامر الأساسية:**
```
!مساعدة                    - عرض قائمة الأوامر
!قائمة_الرومات             - عرض كل الرومات المعدة
!إعدادات_الروم [اسم]       - عرض إعدادات روم معين
```

### ⚙️ **أوامر المشرفين:**
```
!تعيين_محتوى [نوع]         - تعيين محتوى للروم الحالي
!مسح_المحتوى [اسم]         - مسح محتوى روم معين
```

## 🎨 **أنواع المحتوى**

### 1. **rules (قوانين)**
```bash
!تعيين_محتوى rules
```
- عرض قوانين السيرفر
- صورة ثلاثية الأبعاد
- قوانين قابلة للتخصيص

### 2. **welcome (ترحيب)**
```bash
!تعيين_محتوى welcome
```
- رسائل ترحيب جميلة
- صور ترحيب متحركة
- رسائل متعددة

### 3. **announcement (إعلانات)**
```bash
!تعيين_محتوى announcement
```
- إعلانات هامة
- صور إعلانية
- نصوص مميزة

### 4. **text (نص عادي)**
```bash
!تعيين_محتوى text
```
- محتوى نصي مخصص
- حقول قابلة للتخصيص
- ألوان مختلفة

## 🎯 **أمثلة الاستخدام**

### **مثال 1: إعداد روم القوانين**
1. اذهب لروم "قوانين" في ديسكورد
2. اكتب: `!تعيين_محتوى rules`
3. الآن عند دخول أي شخص للروم الصوتي، سيظهر القوانين في روم القوانين

### **مثال 2: إعداد روم الترحيب**
1. اذهب لروم "ترحيب" في ديسكورد
2. اكتب: `!تعيين_محتوى welcome`
3. سيظهر رسالة ترحيب عند دخول الروم الصوتي

### **مثال 3: تخصيص محتوى**
1. استخدم مدير المحتوى: `python content_manager.py`
2. اختر "إضافة قناة جديدة"
3. حدد النوع والمحتوى المخصص

## 🔧 **تخصيص المحتوى**

### **تعديل الصور:**
في مدير المحتوى، غيّر روابط الصور:
```json
{
  "image": "https://i.imgur.com/your-custom-image.png"
}
```

### **تعديل الألوان:**
استخدم أرقام الألوان HEX:
```json
{
  "color": 5814783  // أزرق
  // 65280           // أخضر
  // 15158332        // أحمر
  // 3447003         // بنفسجي
}
```

### **إضافة حقول مخصصة:**
```json
{
  "fields": [
    {
      "name": "📌 مهم",
      "value": "نص مخصص هنا",
      "inline": false
    }
  ]
}
```
