# 🤖 بوت القوانين الذكي

بوت ديسكورد متقدم لعرض القوانين تلقائياً عند دخول المستخدمين للرومات الصوتية وحذفها عند الخروج.

## 🌟 المميزات

### 📜 نظام القوانين المتقدم
- **عرض تلقائي** للقوانين عند دخول الروم الصوتي
- **حذف تلقائي** للقوانين عند خروج المستخدم
- **تخصيص كامل** للقوانين من لوحة التحكم
- **صورة قوانين 3D** قابلة للتخصيص

### 🎛️ لوحة تحكم متكاملة
- **إدارة القوانين** (إضافة، تعديل، حذف)
- **تخصيص الألوان والصور**
- **إحصائيات القنوات النشطة**
- **إعدادات متقدمة**

### ⚡ تقنيات متقدمة
- **WebSocket** لمراقبة دخول/خروج الرومات
- **Redis** للتخزين المؤقت (اختياري)
- **Flask** للواجهة الخلفية
- **استجابة فورية** للأحداث

## 🚀 التثبيت والنشر

### 1. المتطلبات
- Python 3.8+
- حساب في Discord Developer Portal
- حساب في Render (للنشر)

### 2. إنشاء تطبيق ديسكورد
1. اذهب إلى [Discord Developer Portal](https://discord.com/developers/applications)
2. أنشئ تطبيق جديد
3. اذهب إلى **Bot** → **Add Bot**
4. انسخ **Application ID** و **Bot Token**

### 3. صلاحيات البوت
البوت يحتاج إلى هذه الصلاحيات:
- `View Channels`
- `Send Messages`
- `Embed Links`
- `Connect` (للرومات الصوتية)
- `Speak` (للرومات الصوتية)

### 4. النشر على Render

#### أ. رفع الملفات
1. ارفع الملفات إلى GitHub أو استخدم رفع الملفات المباشر
2. أنشئ **Web Service** جديد في Render

#### ب. إعدادات البناء
- **Build Command:** `pip install -r rules_requirements.txt`
- **Start Command:** `python rules_bot.py`
- **Runtime:** Python 3

#### ج. متغيرات البيئة
أضف هذه المتغيرات في Render:
```
DISCORD_TOKEN = توكن البوت
APPLICATION_ID = معرف التطبيق
GUILD_ID = معرف السيرفر (اختياري)
ADMIN_PASSWORD = كلمة مرور لوحة التحكم
FLASK_SECRET = أي قيمة عشوائية
REDIS_URL = redis://your-redis:6379 (اختياري)
```

## 🎛️ لوحة التحكم

بعد تشغيل البوت:
1. افتح الرابط الذي يوفره Render
2. سجل الدخول بكلمة المرور
3. استخدم لوحة التحكم لإدارة القوانين

### الميزات المتاحة:
- **تبويب القوانين:** إضافة، تعديل، حذف القوانين
- **تبويب القنوات:** إدارة القنوات النشطة
- **تبويب الإعدادات:** تخصيص المظهر والرسائل

## 🔧 الإعدادات المتقدمة

### تخصيص القوانين
```json
{
  "welcome_message": "مرحباً بك في السيرفر! يرجى قراءة القوانين أدناه:",
  "rules_image": "https://example.com/rules.png",
  "embed_color": "#667eea"
}
```

### قنوات القوانين
يمكن تفعيل القوانين في عدة قنوات صوتية مختلفة
كل قناة لها إعداداتها الخاصة

## 🐛 حل المشاكل

### البوت لا يعمل
1. تحقق من توكن البوت
2. تأكد من الصلاحيات الممنوحة للبوت
3. تحقق من متغيرات البيئة

### لا تظهر القوانين
1. تأكد من تفعيل القناة في لوحة التحكم
2. تحقق من أن البوت لديه صلاحية إرسال الرسائل
3. تأكد من أن المستخدم دخل الروم الصوتي

### مشاكل WebSocket
1. تحقق من اتصال الإنترنت
2. أعد تشغيل البوت
3. تحقق من سجلات الأخطاء

## 📞 الدعم

إذا واجهت أي مشاكل:
1. تحقق من سجلات البوت في Render
2. تأكد من إعدادات الصلاحيات
3. تحقق من متغيرات البيئة
